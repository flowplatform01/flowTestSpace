# Flow Test Space

This repository serves as a testing space for various Flow ecosystem test applications and integration testing suites.

## Structure

Each test application resides in its own subdirectory:
- `flowpay-external-test-app` - External integration testing for FlowPay

## Purpose

This space provides a centralized location for:
- E2E testing suites
- Integration testing applications
- External API test tools
- Payment gateway sandbox testing

Each test application is self-contained with its own dependencies and configuration.

## Usage

Navigate to individual test application directories for specific setup and usage instructions.
